# engineering-principles

## 기술부채
- 기술부채란 무엇일까?
- 어떻게 하면 기술부채를 줄일 수 있을까?
- 기술부채를 해소하는 일들은 기능개발과 트레이드오프일까? 기술부채를 해소하면 기능개발속도가 느려지는가?
